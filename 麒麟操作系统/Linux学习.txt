基础：

一，命令（每个命令后都有空格）-r表示文件夹       -f表示强制(做什么事情都没提示)
1, /						表示层次等级 
2，ls               				列出文件（普通款）
3，ls -a   					表示all的意思，列出所有的目录（包括隐藏文件）
4，ls -l     					以列表的形式去展现内容（展示更多的信息）
5，ls -a -l
6，ls -al					（若在后面加上“ /”的话，会激发列出隐藏的文件内容）
7，ls -la					都是代表以上内容综合列出来
8，ls -lh					代表列出文件的所有内容的同时，还列出相应内容的内存大小
9，cd 加Linux路径			（chage directory）更改目录
10，cd /					指切换到根目录下
11，cd					自动回到根目录下
12，pwd   				（print work diretory）打印当前的工作目录
13，mkdir 加创建的名称		（make diretory）创建目录（文件夹），如mkdir document
14，touch 加创建的名称		用来创建文件
15，cat 加Linux路径			用来查看文件夹的内容  
16，more 加Linux路径		用来查看文件夹内容
					cat与more的区别：cat是查看所有内容，而more是翻页找内容
					more适用于多文件夹的文件内容查找，点空格即可翻页，按q退出
17,cp （-r）被复制的路径 复制路径去的地方	 （copy）复制文件/文件夹，-r代表复制文件夹
       mv	              c1                  c2
	被移动 的文件 移动到的地方		 （move）移动文件/文件名
	需改名的文件 改成的文件名		   改名（c2名字必须不存在，才能改名，否移动）
18,rm -r -f 参数1 参数2........参数n		   （remove）用于文件/文件夹的删除，-f表示									   force（强制删除文件/文件夹，不会给提示）
19,*				表示通配符（模糊匹配）    test*则匹配有关test开头的内容	
										*test则匹配有关test结尾的内容
										*test*则匹配所有有关test的内容	
20,su - root				再输入密码(不显示)后进入高级管理员身份，输入exit则退出
21,which 加命令			查找命令的程序文件位置
22,find	起始路径 -name “被查找的文件名”
					查找文件，搜索文件
23,find 起始路径 -size +|- n【kMG】
					查找规定大小的文件
				如： find / -size + 10k      查找大于10k的文件
					find / -size - 20M	查找小于20MB的文件	
					find / -size + 50G	查找大于50G的文件
24,grep 【-n】 “关键字“ 文件路径
					通过关键字查找过滤行
				  如，grep ”test“ /         查找包含test关键字的内容
					grep -n ”test“  /	查找包含test关键字的内容在第几行
25,wc 【-c -m -l -w】 路径	统计文件的数量等
					(-c统计byte数量-m统计字符数量-l统计行数-w统计单词数量)
				   如，wc test          得到结果为	 行数 单词数 字节数 文件名
					  wc -c test      得到结果为  test的byte数量
 26,  |   					管道符，左边结果作为右边结果的输入(可嵌套使用)
			        如，cat test | grep huang  通过test来查找过滤出huang
27,echo  ”输出内容“		输出指定内容
     echo`内容`				飘号（反引号）执行内容里的命令
				   如，echo `pwd`    就会打印出pwd的命令内容 
	重定项符：
     echo ”内容“>文件名       将内容输入文件内，并覆盖原本文件里的内容
     echo ”内容“>>文件名	将内容追加到文件内，不覆盖原本文件里的内容
28,tail [-num -f] 文件路径		查看文件的尾部内容(默认10行)    -f 表示持续追踪的意思(实时更新文件尾部的内容)             按Ctrl+C退出命令
